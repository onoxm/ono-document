# 快速开始

## ono-react-element

### 介绍
ono-react-element 是一个基于 React 的组件库。

### 安装

#### 前置准备
- <a href="https://zh-hans.react.dev/" target="_blank">React</a> 18 及以上版本。

::: code-group
```bash [npm]
npm install ono-react-element
```
```bash [yarn]
yarn add ono-react-element
```
```bash [pnpm]
pnpm add ono-react-element
```
```bash [bun]
bun add ono-react-element
```
:::

### 引入样式
在 React 项目入口文件中引入样式：

```tsx
// src/main.tsx
import 'ono-react-element/dist/index.css'
```

## @onoxm/vite-plugin-auto-router

### 介绍
@onoxm/vite-plugin-auto-router 是一个基于 Vite 的插件，用于自动生成 React 或 Vue 路由文件的 Vite 插件。

### 安装

#### 前置准备
- <a href="https://cn.vitejs.dev/" target="_blank">Vite</a> 7.0 及以上版本。

::: code-group
```bash [npm]
npm install -D @onoxm/vite-plugin-auto-router
```
```bash [yarn]
yarn add -D @onoxm/vite-plugin-auto-router
```
```bash [pnpm]
pnpm add -D @onoxm/vite-plugin-auto-router
```
```bash [bun]
bun add -D @onoxm/vite-plugin-auto-router
```
:::
